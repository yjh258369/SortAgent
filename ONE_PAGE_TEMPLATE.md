# Claude Code 任务编排 - 单页速查表

---

## 🎯 核心任务提示词

```
整理 [目录] 的 .md 文件：
1. 生成 200 字摘要
2. 打标签（时间/类型/重要性 1-5星）
3. 识别重复内容
4. 创建 README 索引
5. 重命名为 YYYY-MM-DD_description.md
6. 删除 <2 星文件
```

---

## 📊 标签系统

| 维度 | 选项 |
|------|------|
| **时间** | 创建日期、更新日期 |
| **类型** | 配置文档、技术日志、会话记录、API文档 |
| **重要性** | ⭐(低) ⭐⭐(一般) ⭐⭐⭐(参考) ⭐⭐⭐⭐(重要) ⭐⭐⭐⭐⭐(关键) |
| **主题** | 根据内容提取（如 MySQL/Docker/API） |

---

## 📁 输出文件

| 文件 | 内容 |
|------|------|
| **README.md** | 文档索引、标签系统、快速查找 |
| **FILE_ANALYSIS.md** | 详细摘要、标签数据、重复分析 |
| **EXECUTION_LOG.md** | 执行记录、统计数据、操作日志 |
| **archive/** | 原始文件备份 |

---

## ⚙️ 参数配置

```json
{
  "target_directory": "./docs/",
  "file_types": [".md"],
  "summary_length": 200,
  "importance_threshold": 2,
  "duplicate_threshold": 0.5,
  "backup_enabled": true
}
```

---

## 🚀 使用场景

| 场景 | 目录 | 重点 |
|------|------|------|
| 技术文档 | ./docs/ | API、配置、教程 |
| 会议记录 | ./meetings/ | 决议、负责人、待办 |
| 项目日志 | ./logs/ | 关键事件、问题、里程碑 |
| 知识库 | ./knowledge/ | 分类、标签、检索 |

## 🚨 快速问题

| 问题 | 解决 |
|------|------|
| 文件太多 | 分批处理，每次 20-30 个 |
| 误删文件 | 从 archive/ 恢复 |
| 标签不准 | 添加自定义标签规则 |
| 链接失效 | 使用相对路径 |

---

**使用方法**：替换 `[目录]` 为实际路径，复制到 Claude Code 执行。
