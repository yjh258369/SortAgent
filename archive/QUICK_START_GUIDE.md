将以下提示词复制到 Claude Code：

```
请帮我整理 /root/.openclaw/workspace/memory/ 目录：

1. 扫描所有 .md 文件
2. 为每个文件生成 200 字摘要
3. 打标签（时间、类型、重要性 1-5星）
4. 识别重复内容
5. 生成 README.md 索引
6. 重命名文件为 YYYY-MM-DD_description.md 格式

完成后生成分析报告。
```

---

## 📋 分步执行

### 步骤 1：生成摘要
```
读取 memory 目录下所有 .md 文件，为每个文件生成 200 字摘要。
```

### 步骤 2：打标签
```
为每个文件打标签：
- 时间：创建日期
- 类型：配置文档/技术日志/会话记录
- 重要性：1-5星
- 主题：OpenClaw/ShellCrash/MySQL等
```

### 步骤 3：识别重复
```
对比所有文件的摘要，找出重复度 >50% 的内容，给出处理建议。
```

### 步骤 4：生成 README
```
创建 README.md，包含：
- 文件列表（按类型和重要性分组）
- 标签索引
- 快速查找指南
```

### 步骤 5：优化结构
```
重命名文件为 YYYY-MM-DD_description.md 格式，删除低价值文件。
```
